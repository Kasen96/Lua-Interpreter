Simple Lua Interpreter
======

An imperfect Lua parser and interpreter.

Follow the [EBNF](http://www.lua.org/manual/5.1/manual.html#8) grammar of Lua 5.1

Code snippets for testing are in the folder of *tests* .

# Environment

1. OS : Ubuntu 18.04
2. graphviz 2.40.1 : `apt-get install graphviz`
3. flex 2.6.4 : `apt-get install flex`
4. bison 3.0.4 : `apt-get install bison`
5. gcc 7.5.0 : `apt-get install build-essential`

# How to use & Commands

1. `make`
2. `./parse ./tests/?.lua`
3. `./gen_dot.sh`
4. `./clean.sh` for deleting all generated files.
5. `make clean` for deleting files generated by flex and bison.

# Notice

* The parser has full functionality, the results are in the svgs folder.
* The interpreter works well on the test cases, it may encounter errors when trying other examples
